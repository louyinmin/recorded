# Travel Accounting Docs

Travel Accounting owns:

- Frontend pages and assets under `projects/travel_accounting/frontend/`.
- Travel APIs implemented in the root Flask entrypoint.
- Public browser routes `/travel-login.html`, `/trips.html`, `/trip.html`, and `/settings.html`.
- API routes under `/api/trips*`, `/api/records*`, `/api/payers`, and `/api/categories`.

The original product request is preserved in `original-requirements.md`.
