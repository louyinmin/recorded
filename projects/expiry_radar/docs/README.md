# Expiry Radar Docs

Expiry Radar owns:

- Frontend pages and assets under `projects/expiry_radar/frontend/`.
- Backend implementation under `projects/expiry_radar/backend/expiry_backend/`.
- Public browser routes under `/expiry/*`.
- API routes under `/api/expiry/*`.
- Reminder and account-management behavior described in `overview.md`.

The root `expiry_backend/` package is only a compatibility entry for existing imports.
