## 5. 数据库设计

### 5.1 表结构

```sql
-- 旅行表
CREATE TABLE trips (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    start_date TEXT,
    end_date TEXT,
    note TEXT,
    created_at TEXT
);

-- 记账记录表
CREATE TABLE records (
    id TEXT PRIMARY KEY,
    trip_id TEXT NOT NULL,
    category TEXT NOT NULL,
    amount REAL NOT NULL,
    payer TEXT NOT NULL,
    date TEXT,
    note TEXT,
    FOREIGN KEY (trip_id) REFERENCES trips(id) ON DELETE CASCADE
);

-- 支付人表
CREATE TABLE payers (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT UNIQUE NOT NULL
);

-- 分类表
CREATE TABLE categories (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT UNIQUE NOT NULL
);
```

### 5.2 默认分类

- 交通工具（飞机/动车/自驾）
- 住宿
- 餐费
- 打车
